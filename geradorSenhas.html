Gerador de Senhas

Ferramentas utilizadas:

  • HTML
  • SASS
  • JavaScript

Este projeto foi desenvolvido como parte do desafio para a vaga de Desenvolvedor Front End na Alura. Foi uma experiência enriquecedora, pois tive a oportunidade de criar um gerador de senhas com validações avançadas, algo que eu ainda não tinha feito antes. O projeto exigiu que eu garantisse que as senhas fossem geradas conforme as especificações definidas, incluindo a implementação de controles de comprimento e características específicas. Além disso, aproveitei para aprofundar meus conhecimentos em SASS, utilizando @mixin para otimizar e reduzir repetições no código. O desafio não apenas testou minhas habilidades técnicas, mas também ampliou meu domínio sobre práticas de desenvolvimento e design.
